# cy-cucumber
[![CI](https://github.com/jeanljh/cy-cucumber/actions/workflows/main.yml/badge.svg)](https://github.com/jeanljh/cy-cucumber/actions/workflows/main.yml)

Example of Cypress Cucumber BDD framework by utilizing npm package ```@badeball/cypress-cucumber-preprocessor```

CI - GitHub Actions to run Cypress tests and GitHub Pages to publish the generated Cucumber HTML report.

GitHub page address to view test report: https://jeanljh.github.io/cy-cucumber/
